2- HTML5

A Linguagem de Marcação de Hipertexto (HTML) é a linguagem principal da web. Conhecida em inglês como Hypertext Markup Language, ela permite que os desenvolvedores desenhem como serão exibidos pelo navegador os elementos da página, como texto, hiperlinks e arquivos multimídia. Essa linguagem funciona de modo estático, o que significa que você não pode criar uma página na internet dinâmica ou interativa usando o HTML. Ele apenas modifica os elementos estáticos de uma página na web, como cabeçalho de conteúdo, rodapé, posição de imagens, etc.
